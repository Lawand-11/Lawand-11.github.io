Masjid Ashty App

A nonprofit Android application developed for the Kurdish Community Islamic Center. Features include live prayer times, event announcements, and donation integration. Built using Kotlin, Jetpack Compose, Hilt, and Room. Published on the Google Play Store.

Nuzlocke Master

A native Android app for tracking Pokémon Nuzlocke challenges. Built with a modern architecture using Kotlin, Jetpack Compose, Room database, and Hilt for dependency injection. Focused on providing an intuitive, fully offline-first experience for dedicated players.

Apple Stock Predictor

A machine learning project developed as part of my Computer Science degree. Built using Python, Streamlit, and scikit-learn, the web app predicts future Apple stock closing prices based on historical data. Emphasizes model evaluation metrics like R² and RMSE.
